# Unloaded-II 0.5.6

This release brings Granblue Fantasy: Relink and Persona 5 Royal onto the same
drop-in version and the same Steam launch option:

```text
WINEDLLOVERRIDES="winmm=n,b" %command%
```

## Downloads

- `reloaded-dropin-gbfr-0.5.6.zip` — Granblue Fantasy: Relink
- `reloaded-dropin-p5r-0.5.6.zip` — Persona 5 Royal

Download the archive for your game and extract it beside the game executable.
The first launch needs internet access so Unloaded-II can download the required
base mods from their authors' official GitHub releases.

## Highlights

- Adds release-ready Persona 5 Royal support under Proton.
- Holds P5R startup until CriFs has counted mod files and installed its hooks,
  fixing the second-launch crash and file-capacity warning.
- Keeps P5R's CriFs bind list valid on clean installs and after removing the
  final user mod.
- Verifies P5R archive import, automatic mod enabling, the INSERT-key overlay,
  and a real Persona Essentials model replacement.
- Retains GBFR cache healing, data-index backup/restore, and mirrored-file
  reconciliation.
- Includes `collect-diagnostics.sh` in both packages for one-step bug-report
  log collection.
- Expands third-party attributions for both game packages and preserves
  Reloaded-II's complete upstream license bundle in each archive.
- Uses one shared `0.5.6` version for both game packages.

Third-party game loaders and libraries are downloaded on first launch and are
not redistributed in these archives.
